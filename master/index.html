![Build Status](https://travis-ci.org/m6w6/ext-propro.svg?branch=master)

The "Property Proxy" extension provides a fairly transparent proxy for internal object properties hidden in custom non-zval implementations.

Documentation

See the online markdown reference.

Known issues are listed in BUGS and future ideas can be found in TODO.

Installing

PECL

pecl install propro

PHARext

Watch out for PECL replicates and pharext packages attached to releases.

Checkout

git clone github.com:m6w6/ext-propro
cd ext-propro
/path/to/phpize
./configure --with-php-config=/path/to/php-config
make
sudo make install
